# Changelog

All notable changes to kryten-misc (echo-bot) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-12-09

### Changed

- **Compatibility**: Lowered minimum Python version requirement from 3.11 to 3.10
  - No Python 3.11+ specific features are used in the codebase
  - Dependencies support Python 3.10

## [0.1.0] - 2025-12-08

### Added

- Initial release of echo-bot example application
- Basic echo functionality using kryten-py library
- Example configuration and setup instructions
